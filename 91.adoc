Daí utilizamos a mesma lógica para incluir os demais valores:

```
(swap! registros conj
                 {:valor 2700M :tipo "receita" :comentario "Bico"
                  :moeda "R$" :data "01/12/2016"})
;; ({:valor 2700M, :tipo "receita", :comentario "Bico",

```
